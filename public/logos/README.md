# Logos

This directory contains stock symbol logos used by the ChuckBot application.

## Usage
- Logos should be in PNG format
- Named with lowercase symbol (e.g., `aapl.png`, `tsla.png`)
- Recommended size: 32x32 pixels
- Transparent background preferred

## Example Structure
```
logos/
├── aapl.png
├── tsla.png
├── msft.png
└── ...
```

## Note
This replaces the external ClearBit API dependency with local logos for better performance and reliability.
